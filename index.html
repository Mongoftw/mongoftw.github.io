Projects Worked On

These are some examples of things that I've worked on while attending Michigan Tech. Some of them are class programs while others are group projects that I participated in.

Stranded at Station 4

Stranded at Station 4 is a digital board game that myself and five others are currently making in Husky Games this semester. It is heavily influenced by the Betayal board game series. My job is UI work right now.

Recolored

Recolored is a puzzle game that myself and 5 others made over the 2017-2018 school year in Husky Games. You would mix different inputs together to recolor the a line that gets submitted. I made all the 3D models and the starting UI

Breaking out Bradly

Breaking out Bradly is a bullet hell genre game that myself and 5 others built over the 2016-2017 school year in Husky Games. I worked on the AI movement

Systems Chat Client

There was a program in my Systems programming class that had us use sockets to make a online chat client. I picked it up after the class to play around with it and add more commands to it.

Concurrent Program

I did 6 programs in my Concurrent Class here at MTU. My favorite was when we had to manage a Mother eagle and her hungry children. They are all threads, and we had to concurrently manage each baby eagle so they didn't upset Mom.

Traffic Counting

The traffic Counting app is a program that myself and 5 others built in our Human Computer Interactions class. We were given a client that wanted a project, and built it over a semester with their constant input. Our client wanted to count traffic.

MyEDU4MTU

MyEDU4MTU was a project that myself and 3 others tried to make in our Team Software Project Class. This project was to try and replace MyEDU which was an online class scheduler that helped college students plan their semesters visully.
