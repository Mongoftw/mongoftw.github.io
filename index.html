Concurrent Program

I did 6 programs in my Concurrent Class here at MTU. My favorite was when we had to manage a Mother eagle and her hungry children. They are all threads, and we had to concurrently manage each baby eagle so they didn't upset Mom.

Traffic Counting

The traffic counting app is a program that myself and 5 others built in our Human Computer Interactions class. The class revolves around you building an application for a client in one semester while keeping them updated on progress and asking for input as the project develops.

MyEDU4MTU

MyEDU4MTU was a project that myself and 3 others tried to make in our Team Software Project Class. This project was to try and replace MyEDU which was an online class scheduler that helped college students plan their semesters visually. In the end, we were way over our head and failed in making a fully working project. We may have failed, but failure is the best teacher because I learned a lot from this project.
